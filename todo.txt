cloudflared code pattern:

# New tunnel or if cert or json file is missing

1) cloudflared tunnel login [run if cert doesnt exist] (creates cert.plm to authenticate tunnel)
2) create config.yml [must create manually]
3) cloudflared tunnel delete KalaServer (deletes old tunnel if exists)
4) cloudflared tunnel create KalaServer (creates new tunnel)
5) cloudflared tunnel route dns thekalakit.com KalaServer (handles context)
6) cloudflared tunnel run KalaServer (runs tunnel)

# Existing tunnel
1) cloudflared tunnel run KalaServer (runs tunnel)
